ACCUEIL

Sur l'écran d’accueil, la connexion est faite automatiquement, cependant si un problème survient, les boutons CUISINE et SALLE sont indisponibles. Le Bouton LOGON vous signale que vous pouvez relancer la connexion. AIDE vous emmène sur cette documentation.

CUISINE

L'écran Cuisine, permet de gérer la création dans la base de donnée d’un nouveau plat ainsi que la gestion du stock en ajoutant ou supprimant un plat disponible.

SALLE

L'écran Salle, permet de gérer les commandes clients avec la possibilité de commander un plat ou d’annuler celui-ci.
